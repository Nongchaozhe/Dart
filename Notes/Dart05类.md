#### 类
```
Dart是面向对象语言，类、单继承。所有类都是Object的子类。类通常包括属性和方法。

- 默认构造函数、自命名构造函数

- Dart没有publish，private，protected这些访问修饰符号
- 可以使用_前缀，将属性和方法定义为私有
- 私有属性/方法，所在类必须单独抽出模块才有效。如果跟调用的地方在同一个文件是无效的，还是可以访问的，只不过参数名字是_xx

- set/get 修饰符
- 构造函数体运行之前初始化实例变量（初始化列表）

```

#### 静态、对象操作符
```
静态成员，静态方法
1、使用static关键字实现类级别的变量和函数
2、静态方法不能访问非静态成员，非静态方法可以访问静态成员

静态方法直接类比成类方法，由类直接调用

对象操作符
?. p?.printInfo(); //p不为空，才调用后面的方法
as 类型转换
is 类型判断
.. 级联操作，连缀，适用场景较多

var stu;
stu='';
stu=new Student('默');
if(stu is Student) {
    print('是学生');  //会打印
}
stu.printInfo();  //直接调用是没问题的，但是代码不会有提醒
(stu as Student).printInfo(); //类型转换下

(stu as Student)..name = "默默"
                ..age = 18;
```

#### 类的继承
```
- 使用extends关键词来继承父类
- 子类会继承父类里可见的属性和方法，但不会继承构造方法
- 子类可以覆写父类的方法、get、set

- super调用父类
```

#### 抽象类
```
主要用于定义标准，子类可以继承抽象类extends，也可以实现抽象类接口implements
- abstract 关键字来定义抽象类
- 抽象方法，不需要abstract声明，没有方法体的方法就是抽象方法
- 子类继承抽象类，必须实现里面的抽象方法
- 如果将抽象类当做接口实现的话，必须实现抽象类里定义的所有属性和方法
- 抽象类不能被实例化，只有继承它的子类可以

多态：父类定义一个方法不识闲，让继承他的子类去实现，每个子类有不同的表现
允许将子类类型的指针赋值给父类类型的指针，同一个函数调用会有不同的执行效果
子类的实例赋值给父类的引用

抽象类也可以当接口，一种约定，规范
- Dart也有借口，但是没有interface关键字定义接口，而是抽象类或普通类都可以作为接口被实现，利用implements关键字来实现
- 如果是普通类，会将普通类和抽象类的属性和方法都全部覆写一遍。一般用抽象类定义接口，因为抽象类只用定义抽象方法就行了
- 一般会将接口和实现分离。接口独立出一个文件

比如数据库有标准，其他的数据库都要实现这一套标准
```
